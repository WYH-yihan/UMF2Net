# UMF2Net

This repository provides the implementation of UMF2Net for hyperspectral anomaly detection.

## Overview

UMF2Net is designed for hyperspectral anomaly detection by combining multi-scale spatial feature extraction, weighted feature fusion, and attention-based feature enhancement.

## Dataset

The experiments are conducted on commonly used hyperspectral anomaly detection datasets, including Airport-1, Pavia Center, Airport-3, and Texas Coast.
